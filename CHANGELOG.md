# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Expo config plugin for Android Glance widgets
- Jetpack Glance integration with Compose-based UI
- React Native bridge via `updateWidgetData()` function
- SharedPreferences-based data persistence
- Automatic Gradle dependency injection
- Automatic AndroidManifest receiver registration
- Automatic MainApplication package registration
- TypeScript support with full type definitions
- Customizable widget colors and layout
- Template files for Kotlin and XML resources
- Comprehensive documentation

### Features
- Display title and subtitle on Android home screen
- Live updates from React Native code
- Minimal configuration required
- Production-ready implementation
- Zero native code required from users
